# Гузариш аз Whereby ба LiveKit — дастури пурра

## Чӣ тағйир ёфт дар index.html

Ҳамаи ҷойҳое, ки пештар Whereby истифода мешуданд (занги хусусӣ, занги видеоӣ, эфири
комната) ҳоло тавассути **LiveKit** (платформаи WebRTC-и кушодаасос) кор мекунанд.
Номи функсияҳо (`createWherebyRoom`, `openWherebyEmbed`, `leaveWherebyRoom` ва ғ.)
қасдан тағйир дода нашуд, то ҳељ ҷои дигари кодатон вайрон нашавад — фақат мантиқи
дохилии онҳо иваз шуд.

Барои кор кардан, се қисмат бояд насб шаванд:

```
Барномаи шумо (index.html)
        │
        │  1. дархости токен
        ▼
Cloudflare Worker (livekit-token-worker.js)
        │
        │  2. токени имзошуда
        ▼
Сервери LiveKit-и худатон (livekit-server/)  ← WebRTC + TURN/TLS дар порти 443
```

## Қадами 1 — Сохтани сервери LiveKit

Ба папкаи `livekit-server/` нигаред. Дастури пурраи насб дар охири
`docker-compose.yml` навишта шудааст. Хулоса:

1. VPS дар Аврупо/Туркия бихаред (на дар Русия).
2. Домен ба он равона кунед, сертификати TLS гиред (Let's Encrypt).
3. Дар `livekit.yaml` калид/сирри API таъин кунед.
4. `docker compose up -d`

## Қадами 2 — Сохтани Cloudflare Worker

Файли `livekit-token-worker.js`-ро ба Worker-и нав дар Cloudflare гузоред
(ройгон аст, мисли Worker-и қаблии Whereby ки истифода мекардед). Дар танзимоти
Worker се "Secret" илова кунед:

- `LIVEKIT_API_KEY`
- `LIVEKIT_API_SECRET`
- `LIVEKIT_SERVER_HTTP_URL` (масалан `https://turn.yourdomain.com`)

## Қадами 3 — Танзими index.html

Дар қисми боло (наздик ба `firebaseConfig`) ду хатро иваз кунед:

```js
const LIVEKIT_WS_URL = "wss://turn.yourdomain.com";
const LIVEKIT_TOKEN_ENDPOINT = "https://СУРОҒАИ_WORKER_ШУМО.workers.dev/token";
const LIVEKIT_END_ROOM_ENDPOINT = "https://СУРОҒАИ_WORKER_ШУМО.workers.dev/end-room";
```

Ҳамин қадар — занги хусусӣ, занги видеоӣ ва эфир бояд бе тағйироти дигар кор кунанд.

## Огоҳии ростқавлона (лутфан хонед)

- Ман коди `index.html`, Worker ва конфиги LiveKit-ро аз рӯйи ҳуҷҷатгузории
  расмии кушодаасоси LiveKit навиштам ва синтаксиси ҳар се (JS ва YAML)-ро
  тафтиш кардам — вале ман **имкони санҷиши воқеӣ дар шабака** (пайвастшавии
  ҳақиқии видео/аудио) надорам, зеро ин ба сервери воқеии шумо ва домен ниёз
  дорад. Пас аз насб, ҳатман бо ду дастгоҳи воқеӣ (масалан як телефон +
  компютер) занг занед ва санҷед.
- Ин ҳал ба ҷои VK Calls SDK интихоб шуд, зеро ҳуҷҷатгузории оммавии VK Calls
  барои ман дастнорас буд ва навиштани код бидуни он хатари бузург дошт (коде,
  ки шояд кор накунад).
- Coturn-и қаблан пешниҳодшуда акнун лозим нест — LiveKit TURN/TLS-и худро дорад.
- Дар Русия/Тоҷикистон/Ӯзбекистон/Қазоқистон санҷиш кунед; агар дар ягон
  минтақа мушкилот бошад, метавонем сервери дуюми LiveKit дар минтақаи дигар
  (масалан Туркия) илова кунем ва барнома аз рӯйи макони корбар интихоб кунад.
